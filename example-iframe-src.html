Reasonable.iframe

Reasonable.iframe is a tiny HTML document that contains a reasonable set of example elements. It was purpose-built for appearing within reasonable.html's example <iframe> element (hello!), which was purpose-built to test a set of classless CSS rules (reasonable.css).

Mozilla recommends that <iframe> elements be styled with CSS, rather than HTML attributes.
